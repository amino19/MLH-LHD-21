### DEPENDENCIES USED

- [Node.js](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/pt-BR/docs/install)

## STARTED WITH..

1. Clone this repository;
2. `cd SpotifyRedesign`;<br />
3. Run `yarn` to install dependencies.<br />
4. Run `yarn ios` or `yarn android`.
